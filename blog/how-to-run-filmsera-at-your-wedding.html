How to Run a Filmsera Film at Your Wedding Reception

Host checklist: Look, QR, memory captain, Reveal.

Filmsera· July 16, 2026 ·9 min read

A wedding Filmsera Film works when the Host does four quiet jobs early: create the Film, lock a Look, print the join path, and set Reveal. Everything else is Guests filling the roll. This is the reception playbook — the same spine as our printable guest photo plan, written as a narrative you can hand to a memory captain.

Terms you’ll use all night: Host, Film, Look, Guest, Shot, Frame, Reveal, Recap.

1. Create the Film (week-of)

Open Filmsera on the Host iPhone and create one Film named for the day — names and date are enough. Set a Quota so the roll feels intentional (24–36 per Guest-heavy reception is a solid default; unlimited quietly turns into a dump).

Filmsera is iOS-only. Android Guests can’t join this Film. If your room is mixed, keep a small stack of physical disposables or a second plan for those tables — don’t pretend the QR covers everyone.

2. Pick one Look and leave it alone

Browse families in Look Lab. Disposable for reception flash and chaos; Instant or 8mm for softer getting-ready and golden hour. Lock one Look for the Film so every Frame feels like the same night.

For a deeper picker — disposable vs VHS vs 8mm — see the field guide in this Journal. Soft-launch the Look at the rehearsal dinner if you can.

3. Print the QR (and say the install part out loud)

Filmsera Guests need the iPhone app. The QR / join code still requires an install — Filmsera does not offer no-download or App Clip guest join today. Competitors like Once lead with that friction reduction; we’re honest about the gap (see Filmsera vs Once). Plan for it:

  • Table cards or a welcome sign with QR + one line: “Scan → download Filmsera → shoot into our Film.”
  • A short mic announcement at welcome or dinner.
  • Seed three to five friends early so the roll isn’t empty when the dance floor starts.

One Film, one Look, one join path. The Host’s job is clarity — not chasing phones all night.

4. Assign a memory captain

Not the couple. One trusted friend who:

  1. Points people to the QR
  2. Helps with the App Store install if someone stalls
  3. Nudges tables to shoot Shots — not just scroll
  4. Texts the Host if the code needs re-sharing

Memory captains are why shared rolls fill. Couples who “hope people find it” get half-empty Films.

5. Set Reveal before the ceremony

Nobody — Host included — should peek at Frames until Reveal. Morning-after is the sweet spot for most receptions: you stay present, then open the roll with coffee. A week later works if you want honeymoon suspense. Ten minutes turns it into a party game; fine for after-parties, usually wrong for vows.

Put the Reveal time on the day-of timeline or a follow-up text so Guests know when to come back.

Day-of Host checklist

Filmsera on Host phone · Look locked · QR printed · memory captain briefed · Guest install mentioned kindly · Reveal set · couple optional “stay offline the roll” · Recap queued for the morning after.

6. Let Guests fill the roll

Your photographer covers the aisle. Guests cover the angles only they have — back tables, bathroom mirror selfies, grandmother by the window, the dance floor after the pro goes home. Don’t hover. Don’t curate mid-night. The Film is the structure; presence is the point.

Optional chapters if you want more than the reception: a second Film for getting ready, or one roll across the whole weekend. Keep the same Host discipline either way.

7. Open every Frame at Reveal — then Recap

When Reveal hits, open the Film together. Then run a Recap: the edit, not the dump. Twelve great Frames beat two hundred unsorted Shots. That’s the roll you’ll actually rewatch next anniversary.
